# notebooks/

Experimental and exploratory scripts that fed into the supported `ross` CLI.

These are **not** production code. They explore different SLAM approaches (MiDaS
depth variants, ORB-SLAM, anchor-based, multi-face reconstruction, etc.) and
are kept as references for the history behind `ross/slam/`.

For the supported workflows, run `ross --help` (or `uv run ross --help`).
Anything here may be broken, require extra dependencies, or diverge from the
current firmware.
